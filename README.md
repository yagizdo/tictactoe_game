# Flutter - Tic Tac Toe Game


## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/tictactoe_game
```

Go to the project directory

```bash
  cd tictactoe_game
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
